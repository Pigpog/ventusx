# ventusx

A free and open-source utility for controlling the advanced features of
the TteSPORTS Ventus X mouse on Linux.

## Usage

Currently, this utility can:

 - toggle the LEDs on the mouse
 - toggle "battle" mode

```
# toggle the palm LED
ventusx palm off
ventusx palm on

# toggle the scroll wheel LED
ventusx scroll off
ventusx scroll on

# toggle "battle" mode
ventusx battle off
ventusx battle on
```

## Compiling

Just run `make`.
